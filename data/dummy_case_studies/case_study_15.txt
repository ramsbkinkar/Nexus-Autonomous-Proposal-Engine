Project Name: Project Athena
Industry: Higher Education
The Client Challenge: A major university's IT helpdesk was inundated by students asking the same questions about course registration, Wi-Fi configuration, and housing policies. Their existing chatbot was a rigid decision-tree that frustrated users and rarely solved complex, multi-part inquiries.
Our Technical Solution: We deployed a state-of-the-art Generative AI RAG solution entirely on-premise to comply with strict student data privacy regulations (FERPA). We utilized open-source HuggingFace models (Mistral-7B-Instruct) hosted on internal GPU clusters via vLLM for high-throughput inference, ensuring no student data was ever sent to external APIs like OpenAI.
We scraped and processed over 10,000 pages of the university's intranet, IT manuals, and syllabus documents. These were embedded and stored in Milvus, an open-source vector database designed for massive scalability. We built an advanced RAG pipeline that utilized "HyDE" (Hypothetical Document Embeddings) to improve retrieval accuracy when students asked poorly phrased or vague questions.
The frontend was a clean React application embedded into the university portal. We also included an analytics dashboard for the IT team to track the most common unresolved queries, allowing them to continually update the vector database with missing documentation.
The ROI & Impact: Handled over 15,000 student queries during the first week of the semester with an 88% successful resolution rate. Completely eliminated the need for external API costs.
Total Project Cost: $290,000
